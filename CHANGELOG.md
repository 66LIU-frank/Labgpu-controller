# Changelog

## v0.1.0-alpha

First alpha release of LabGPU as a personal GPU workspace for students using shared SSH GPU servers.

- Personal training workflow: find GPU, run or adopt, observe, diagnose, export context, and safely act on your own processes.
- Agentless SSH Home for GPU, process, disk, load, and alert visibility across selected SSH hosts.
- Train Now GPU recommendations with copyable SSH, `CUDA_VISIBLE_DEVICES`, and launch snippets.
- `labgpu pick`, `labgpu where`, `labgpu diagnose`, `labgpu context --copy`, and fake-lab demo flows.
- Cached-first UI pages so navigation does not block on slow SSH probes.
- Chinese/English and light/dark browser toggles.
